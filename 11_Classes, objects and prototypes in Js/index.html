Classes & Objects

Classes

A class is like a blueprint for creating objects. It defines properties(variables) and methods(functions) that the objects created from the class will have.

Object

An objectv is a real instance of a class
It has actual values for the properties define in the class.

Summary of terms

class :- blueprint of objects
objects :- instance of class
constructor :- special method to initialize object properties
Method :- Function define in the class

Prototypes in Js

A JavaScript object is an entity having state and behaviour (properties and mehods)
We can set prototype using __proto__

Inheritance in Js

Inheritance allow one or more child clasees to use properties and method of another parent class.
In JavaScript Inheritance is mainly done using classes and extends keywords.

Syntax:

class parents {
}
class child extends parent {
}

Summary

✦ Use extends for class inheritance
✦ Use super() for call parent constructor
