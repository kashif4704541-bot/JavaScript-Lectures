Events in Js

An event is something that happen on a webpage. Like a button being clicked, a key being pressed or the page loading

JavaScript let you "listen" for these events and runn some code when they happen

Most Common Events

  • click
  • mouseover
  • mouseout
  • keydown
  • Submit
  • load

Event handling in Js

node.event = () =>{
 //Handle here
}

Event Objects:-

It is a special type of object that has details about the event

All event handlers have access to the event object's properties and method
 node.event =(eventName) =>{
   //handle here
 }
e.target,e.type,e.clientX,e.clientY.

Event listeners:-

An event in Js lets you run some code when an event (like a hover, key press, click etc) happen on an element

node.addEventListener(event, callback)
node.removeEventListener(event, callback)
