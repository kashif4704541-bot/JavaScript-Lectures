DOM manipulation

Attributes

You can manipulate HTML Attribute using the DOM. Here's how you can get, set,remove and check attribute of element

1) Get an Attribute:-

.getAttribute("attributeName") // to get the attribute value

2) Set an Attribute:-

.setAttribue("AttributeName", "value) //to set the attribute value

3) Remove an attribute:-

.removeAttribute("AttributeName") //to remove an attribute

Check if Attribute exist:-

.hasAttribute("AttributeName")

Insert Element

In Js, you can insert a new element but first have to create it using Js

i) Create a new Element

let newEl = document.createElement("p");
newEl.textContent = "This is a new paragraph"

ii) Insert Method:-

  • appendChild():-
    • Add a child at the end of parent Syntax: document.parent.appendChild(newEl)
  • append()
    • Just Like appendChild() but can also Add textnodes and multiple elements, Syntax: parent.append("newEl", "Some text")
  • prepend():-
    • Insert at the beginning of parent Element, Syntax: parent.prepend(newEl)
  • before():-
    • Insert before a specified element, Syntax: target.before(newEl)
  • after()
    • Insert after a specified element, Syntax: target.after(newEl)
  • insertBefore():-
    • Insert a node before a reference child of a specified parent, Syntax: parent.insertBefore(newEl, existingChild)
