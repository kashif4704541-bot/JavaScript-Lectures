Error Handling :-

Error handling in Js is done using the try...catch block. It allows you to gracefully handle the errors instead of letting your code crash.

Syntax

try{
    ...normal code
}catch(err){
    ...handling error
}

Sync in JavaScript

Synchronous:-

Synchronous in javaScript means, code is executed line by line in order. Each task must finish before the next start.
  If one line takes time the entire program waits for it to complete.Example...

console.log("Task 1")
console.log("Task 2")
console.log("Task 3")

Asynchronous:-

(Async) programming allows multiple tasks to run without waiting for eachother
  It let javaScript handle long running operation (like API calls, timer, file reading) without blocking the rest of code.Example...

console.log("START");
setTimeout(() =>{
    console.log("Assign task")
}, 2000);
console.log("END")

OUTPUT:
START
END
asign task
