What is DOM ?

DOM stands for Document Object Model. It is a programming interface for web documents. The DOM represent the structure of an HTML Document as a tree of object, which allow programming language like CSS and JavaScript to interact with and manipulate the content, structure and style of web page..

DOM Manipulation

DOM manipulation means using JavaScript to access, change, add or remove the elements or the content of web page

Accessing The Element

To manipulate elements on webpages using JavaScript you must access or select them first. The DOM gives you several ways to do that
