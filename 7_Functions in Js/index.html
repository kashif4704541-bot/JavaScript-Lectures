Function in JavaScript

Function are block of code that perform specific task. They can be used multiple time through out the program.

i) Function defination:

Syntax: function function Name(){
      //block of code
}

ii) function call:

      Syntax: functionName()

Arrow Function

Arrow function are shorter and cleanr way to write code introduced in ES-6
SYNTAX:
const functionName(param1, param2,...)=>{
      //block of code
}
